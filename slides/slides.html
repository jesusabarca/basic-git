Viewing changes

  • Unstaged changes
$ git diff
  • Staged changes
$ git diff --staged

Commiting changes

(snapshots)

  • With text editor
$ git commit

$ git config --global core.editor nvim
  • Inline comment
$ git commit -m 'An awesome commit message.'
